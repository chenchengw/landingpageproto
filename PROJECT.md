# OPSA Landing Page Rebrand Project

## Goal
Rebrand and improve the OPSA landing page (opsa.ai), drawing inspiration from InstaLILY (instalily.ai) while maintaining OPSA's unique value proposition for wholesalers and manufacturers.

## Project Structure

```
landingPage/
├── PROJECT.md              ← You are here — project overview
├── context/                ← Drop OPSA brand assets, decks, screenshots here
├── research/               ← Competitive analysis & strategy docs
├── prototypes/             ← HTML prototypes of new landing page designs
└── .skills/
    └── skills/
        └── landing-page-designer/   ← Claude skill for generating layouts & HTML
```

## Workflow

### Phase 1: Research & Strategy
1. Review the competitive comparison in `research/comparison-opsa-vs-instalily.md`
2. Add any OPSA-specific context (investor deck, brand guide, etc.) to the `context/` folder
3. Discuss and select a strategic direction for the rebrand

### Phase 2: Design & Prototyping
1. Use the `landing-page-designer` skill to generate layout proposals
2. Review HTML prototypes in `prototypes/`
3. Iterate on the selected design with feedback

### Phase 3: Finalization
1. Polish the final HTML prototype
2. Extract design specs for handoff to dev team
