# OPSA Context Folder

Drop any files related to OPSA into this folder so Claude can reference them during the landing page redesign project. Useful items to add:

- **Investor deck** (PDF/PPTX) — messaging, positioning, metrics
- **Brand guidelines** — logos, colors, fonts, tone of voice
- **Competitor screenshots** — other AI sales tools, B2B SaaS sites
- **Customer testimonials / case studies** — quotes, logos, results
- **Product screenshots / demos** — UI mockups, dashboard views
- **Analytics data** — current site traffic, bounce rates, conversion funnels
- **Sales collateral** — one-pagers, brochures, email templates
- **Domain-specific content** — industry reports, market research

Claude will automatically read files from this folder when working on landing page tasks.
